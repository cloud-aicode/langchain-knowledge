# LangChain 1.0 知识点整理

基于 LangChain 1.0 Python 官方文档整理的知识点教程。

## 在线预览

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/cloud-aicode/langchain-knowledge)

## 本地开发

1. 安装依赖：
   ```bash
   npm install
   ```

2. 启动预览：
   ```bash
   npm start
   ```

3. 访问 http://localhost:3000

## 目录结构

- `index.html`: 首页
- `assets/`: 静态资源 (CSS, JS)
- `api-reference/`: API 文档
- `tutorials/`: 教程章节 (01-18)

## 部署

本项目配置为静态网站，可直接部署到 Vercel、Netlify 或 GitHub Pages。
